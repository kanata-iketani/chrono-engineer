# Chrono Engineer（仮）
 
## 特徴
 
- ストーリー主導のアドベンチャー（ビジュアルノベル ＋ 選択体験）
- 遊んでいるうちに、歴史・理数・ものづくりの考え方が自然と身についていく「気づいたら学んでいた」体験
- 教育の押し売りはしない。あくまで物語を追う面白さが主役
- 日本史を縦に旅していく構造（今後、時代を追加して広がっていく予定）
## 技術スタック

- ゲームエンジン: Unity
- 言語: C#
- ターゲット: Steam ＋ スマートフォン（iOS / Android）
## 開発状況

🚧 開発初期。冒頭〜第一章（体験版）を制作中。
